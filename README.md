# tiger-tool
虎码-RIME中州韵助手（rimetool)版                  

# 1、拼音反查
  - F 引导
# 2、以形查音
  - ~ 引导
# 3、精准造词                                                           
  - ` 引导
# 4、计算器
  - = 引导
# 5、日期数字大小写
  - 任意字母大写 引导
# 6、表情
  - Ctrl + Shift + Y
# 7、Unicode编码
  - Ctrl + Shift + I
# 8、Unicode分区           
  - Ctrl + Shift + O                                             
# 9、字、字词模式
  - Ctrl + Shift + G
# 10、拆分      
  - 拆分 + code + 拼音
    - Ctrl + Shift + H     
  - 保留拆分 + 余码                                               
    - Ctrl + Shift + J
  - 只保留拼音            
    - Ctrl + Shift + P         
# 11、繁简
  - Ctrl + Shift + F
